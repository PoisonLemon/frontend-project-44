Brain Games
Математическая игра для тренировки в математике. Включает в себя несколько игр в стиле вопрос-ответ.

Требования:
Node.js версии 14 или выше (тестировано на v18+).
npm (поставляется с Node.js).

Установка:
1. Клонируйте репозиторий или скачайте файлы проекта.
2. Перейдите в директорию проекта:
    cd frontend-project-44
3. Установите зависимости:
    make install

Запус:
Всего присутствует 5 игр, по три раунда каждая, их можно вызвать с помощью команды:
    brain-(название нужной вам игры)

Игры:

even - Ответьте "yes", если число четное, "no" - если нечетное.
[View brain-even demo](https://asciinema.org/a/hAN6W7sqU7JAfds2pzMGU2zsv)

calc - Решите простые арифметические выражения (сложение, вычитание, умножение).
[View brain-calc demo](https://asciinema.org/a/WMqzx31CrMr7AiUd6XWCd7VEj)

gcd - Найдите наибольший общий делитель двух чисел.
[View brain-gcd demo](https://asciinema.org/a/6NZAkRKPcCtj5CpXw6RLqEAKs)

progression - Определите пропущенное число в арифметической прогрессии.
[View brain-progression demo](https://asciinema.org/a/Y0NA88f4mT80fyJ3IvHWSji0I)

prime - Ответьте "yes", если число простое, "no" - если нет .
[View brain-prime demo](https://asciinema.org/a/vfjnNoDNHdBEXp6uzhm2t9LyN)
